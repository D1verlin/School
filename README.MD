## Технологии

- **Frontend**: React, Vite, Tailwind CSS, Framer Motion (анимации).
- **Backend**: Node.js, Express.
- **База данных**: SQLite (библиотека `sql.js`, база хранится в файле `server/school.db`).
- **Аутентификация**: JWT (JSON Web Tokens).

---

## 🛠 Установка и запуск

Для работы приложения необходимо наличие [Node.js](https://nodejs.org/) (рекомендуется версия 18+).

### 1. Клонирование и установка зависимостей

Установите зависимости для фронтенда и бэкенда:

```bash
# Установка зависимостей фронтенда (в корневой директории)
npm install

# Перейдите в директорию сервера и установите его зависимости
cd server
npm install
cd ..
```

### 2. Заполнение базы данных (Seeding)

Перед первым запуском необходимо создать и наполнить базу данных тестовыми данными (учителя, ученики, оценки, расписание):

```bash
node server/seed.js
```
*После выполнения в папке `server/` появится файл `school.db`.*

### 3. Запуск приложения

Приложение состоит из двух частей, которые нужно запустить одновременно (в разных терминалах):

#### Запуск Бэкенда (API)
```bash
cd server
npm run dev
```
*Сервер запустится на [http://localhost:3000](http://localhost:3000)*

#### Запуск Фронтенда (UI)
```bash
# В корневой директории
npm run dev
```
*Приложение будет доступно по адресу [http://localhost:5173](http://localhost:5173)*
